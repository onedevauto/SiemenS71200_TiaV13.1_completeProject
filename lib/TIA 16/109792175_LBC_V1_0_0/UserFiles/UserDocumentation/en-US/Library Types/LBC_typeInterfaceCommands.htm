LBC_typeInterfaceCommands (UDT)

Description

Module related commands from external systems

Parameter description

IdentifierData TypeDefault ValueDescription
refreshConfigurationBoolFALSETRUE: Copy global configuration parameters into module interface for external systems
editConfigurationBoolFALSETRUE: Edit mode for configuration parameters in control system is active - copying into module interface is stopped
saveConfigurationBoolFALSETRUE: Save configuration parameters from module interface into to global configuration parameters after change
acknowledgeBoolFALSETRUE: Module error acknowledge / reset of errors in function block
