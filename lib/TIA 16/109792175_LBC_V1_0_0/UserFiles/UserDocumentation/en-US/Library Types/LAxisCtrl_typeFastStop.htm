Parameter description

IdentifierData TypeDefault ValueDescription
decelerationLReal-1.0Deceleration for fast stop (Value > 0.0: The specified value is used; Value <= 0.0: emergency deceleration of
the technology object is used)
jerkLReal-1.0Jerk for fast stop (Value > 0.0: Smooth velocity profile, the specified value is used; Value = 0.0: Trapezoid velocity profile; Value < 0.0: Use of the default value configured in the technology object)
abortAccelerationBoolFALSETRUE: The acceleration is set to 0.0 at the start of the job, and the deceleration immediately builds up; FALSE: The current acceleration at the start of the job is reduced using the configured jerk. Afterwards, the deceleration builds up
replaceStopPossibleBoolFALSETRUE: Fast stop command can be replaced by another command
